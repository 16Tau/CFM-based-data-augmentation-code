
# CFM_code

Hello,

This supplementary material contains the source code for all the experiments mentioned in the paper. All the experimental results in the paper can be reproduced by executing the source code files.

The file `constants.py` contains the configuration parameters used in the code.

This supplementary material mainly includes seven folders. The name of each folder corresponds to the title of the corresponding content in the paper.

# Dataset
Dataset used in the paper is available at https://www.kaggle.com/datasets/zhiyuanpiao/50subset and https://www.kaggle.com/datasets/zhiyuanpiao/rscd-10.
